# ============================================================================
# SYSTEM REQUIREMENTS
# ============================================================================

# Minimum Requirements:
# - Python 3.8+
# - 4 GB RAM (8 GB recommended)
# - 50 GB free disk space for complete textbook collection
# - Git 2.30+ (system dependency)
# - Internet connection for repository discovery

# Recommended Configuration:
# - Python 3.11+
# - 16 GB RAM for large collections
# - SSD storage for optimal performance
# - Multi-core CPU (system auto-detects and uses available cores)
# - Stable internet connection for parallel downloads

# Performance Notes:
# - Parallel processing automatically scales to available CPU cores
# - Memory usage scales with collection size (~50 MB per 100 textbooks)
# - Processing speed: ~10-50 textbooks per minute depending on source format
# - Books directory excluded from git repository due to size (16GB when populated)

# Optional Features:
# - BeautifulSoup4: Enhanced HTML/XML parsing for some discovery features
# - Anthropic + PyPDF2: PDF processing with Claude API for text extraction
# - All core functionality works without optional dependencies
